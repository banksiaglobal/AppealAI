Application:

ApppelAI is a smart assistant that helps you write an appeal against an insurance company's refusal to pay for hospital procedures.

It is project was created as a technology example to participate in InterSystems Vector Search, GenAI and ML Contest , it was created to demonstrate how our team uses Vector Search and GenAI tools in our solutions.

Technologies used:

Frontend - Angular

Backend - IRIS + LangChain + FHIR
